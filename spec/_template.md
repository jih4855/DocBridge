## 10. 보안 고려사항 (Security Considerations)
- **Path Traversal 방지:** 사용자 입력 경로가 허용된 범위를 벗어나 시스템 중요 파일에 접근하지 않도록 검증.
- **XSS 방지:** 마크다운 렌더링 시 악성 스크립트가 실행되지 않도록 Sanitization 처리.
- **접근 제어:** 로컬 호스트 전용 서비스이므로 허용되지 않은 외부 IP에서의 접근 차단 (CORS 설정 등).
- **심볼릭 링크 처리:** 외부로 연결된 심볼릭 링크를 무시하거나 엄격히 검증하여 의도치 않은 경로 노출 방지.
